+ [prog0903b.m](./非线性回归/prog0903b.m) 姜启源p328 酶促反应问题，Michaelis-Menten模型，混合反应模型及其简化，非线性回归
+ 姜启源p326 首先是将Michaelis-Menten模型线性化，试图通过线性回归估计参数，但效果欠佳，以此还是采用了非线性回归；
